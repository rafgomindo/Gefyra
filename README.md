**Gefyra (Γέφυρα):** The definitive and indestructible bridge between **Zotero** research libraries and the world's most advanced **AI Agents**. Built for **Doctoral Research** and high-integrity academic data management.

Developed by **Rafael Domingo Ramones**, Gefyra (Greek: Γέφυρα) is the "Steel Heart" (Bakal) of modern research workflows.

## 🤖 The Universal AI-Agentic Bridge
Gefyra provides a seamless interface for all major Large Language Models and IDE-integrated agents:
*   **Gemini (Antigravity/CLI)**: Full native integration. 🚀
*   **Claude (Desktop/Dev)**: High-speed metadata retrieval.
*   **ChatGPT (Custom GPTs)**: Via standard MCP relay.
*   **Moxie / Cursor / Windsurf**: IDE-integrated research assistant connectivity.

## 🚀 Autonomous Maintenance Suite (Architect Grade)
Gefyra v3.0.0 is more than a bridge—it is an **Autonomous AI Librarian** capable of self-healing your database:

*   **🧬 Metadata Rescue (DOI Auto-Fetch)**: Scans "Untitled" or broken fragments, extracts DOIs from PDF content, and retrieves elite-level metadata (Title, Author, Journal, Year) via **CrossRef**.
*   **🧱 Structural Part-Fusion**: Intelligently identifies and merges split volume segments (e.g., `Partie 1`, `Partie 2`) into a single, professional Master Entry.
*   **🔭 Zero-Orphan Protocol**: A deep audit system ensuring 100% attachment parenting across the library.
*   **📐 Barbaric Title Remediation**: Detects and fixes "paragraph-length" title errors, restoring clean, accessible record names.

## 🛠️ Main Features
-   **Deep Library Search**: Find any paper or law by title, author, or keyword.
-   **BibTeX Generation**: Instant citation generation for professional academic writing.
-   **Multi-Platform Sync**: Support for **Local Zotero Desktop** (via HTTP) and **Zotero Cloud API** with seamless failover.
-   **Auto-Audit Tagging**: Every fix is tagged with `gefyra:fixed` or `gefyra:metadata-fetched` for total library auditability.
